# 域名申请免费证书

  1. 申请域名：lengband.wang
  2. 为域名申请免费证书，注册登录 https://freessl.cn/
  3. 点击 创建免费的SSL证书
  4. ACME 域名配置（参考：https://blog.freessl.cn/acme-quick-start/）
  5. 执行脚本后如下图，cert和key在下面地址，可以直接用在nginx等服务器中
  6. 配置成功的会显示在这里
  7. 本地可以通过配置/etc/hosts的方式验证 127.0.0.1 www.lengband.wang
  8. 以nginx服务器为例，配置ssl_certificate 和 ssl_certificate_key，启动 nginx
    ssl_certificate      /Users/oker/.acme.sh/www.lengband.wang/www.lengband.wang.cer;
ssl_certificate_key  /Users/oker/.acme.sh/www.lengband.wang/www.lengband.wang.key;
